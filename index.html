Setup

One config block,
instant project awareness.

No API keys, no accounts. Install globally, run pm init, and your AI instantly sees project context.

terminal
# Install globally
$ npm install -g @gida-concept/pm-agent-cli

# Initialize in your project
$ cd your-project && pm init
│
│ Config: .pm-agent/config.toml
│ Database: .pm-agent/pm.db
│ Rules: .pm-agent/rules.toml
│ CLAUDE.md: CLAUDE.md

# Check project state
$ pm status
│ Found 2 blockers, 4 decisions, 3 active rules
1

Install

Node.js v18+. npm install -g globally, or use npx @gida-concept/pm-agent-cli.

2

Init

Run pm init in your project. Creates config, database, rules, and Claude Code hooks.

3

Restart

Restart your editor or CLI session. Claude Code hooks load automatically.

4

Go

Ask naturally: “What’s blocking me?” — PM Agent handles the rest.
