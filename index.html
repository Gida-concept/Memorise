Features

Everything Your AI Needs
to Manage Projects

PM Agent gives your coding assistant structured awareness of your project's state — not just code, but context.

Decision Tracking

Log architectural and project decisions as ADRs with full rationale. Your AI can query, link, and enforce them.

Blocker Management

Surface, track, and resolve blockers. Your AI knows what's blocking progress and can suggest unblock strategies.

Task Awareness

Your AI reads your task list and sprint scope — it understands what's in flight before suggesting next steps.

Dependency Mapping

Analyze how decisions, tasks, and code modules interconnect. Know the blast radius before making changes.

Scope Analysis

Run scope checks on new feature requests against sprint capacity. Your AI will say "this fits" or "too risky."

Rules Engine

Define PM rules (scope, code, process) with hard or soft enforcement. Your AI catches violations before they land.

Codebase Scanning

Scan your codebase for affected modules when a decision changes. Your AI connects PM context to source code.

Project Memory

Persistent memory that survives context limits. Your AI remembers past decisions, notes, and project state across sessions.

MCP Tools

17+ Tools Your AI Can Call

Every tool follows the Model Context Protocol. Your AI chooses the right one — you just talk naturally.

pm_get_context Project overview — blockers, decisions, tasks, scope
pm_log_decision Record an architectural or process decision
pm_add_blocker Log a new blocker with description and severity
pm_resolve_blocker Mark a blocker as resolved
pm_add_note Save an observation or insight
pm_scope_check Check if a feature fits current sprint capacity
pm_add_rule Add a PM rule (scope / code / process)
pm_scan_codebase Scan code affected by a decision or change
pm_analyze_dependencies Map dependency chains between tasks and code
pm_get_timeline Activity timeline for recent project changes
pm_get_rules List all active PM rules
pm_check_rules Check rules against a proposed action
Getting Started

Set Up in 4 Steps

Works with every major AI coding tool. No API keys, no accounts — just a config file.

1
Install Node.js

PM Agent runs on Node.js v18+. No global install needed — npx handles everything.

2
Add Config

Drop the correct JSON / TOML snippet into your AI tool's MCP config file.

3
Restart Your AI

MCP config is read at startup. Restart your editor or CLI to pick up the changes.

4
Start Using It

Just ask your AI naturally — "What are my blockers?" — and PM Agent handles the rest.
